# Nosus

The blog platform without surprises

## Getting Started

```
npm ci
```

## Development

Run the development server: `npm run dev`.
Dev available on: `http://localhost:3000`.

## Production

Run the build script: `npm run build`.

## Serve

Run the serve script: `npm run serve`.
